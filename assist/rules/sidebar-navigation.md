# Sidebar Navigation Rules

## Organisation Accordion Lock

1. The desk sidebar must keep `Overview` as a standalone top-level row linked to `/desk`.
2. `Overview` must not show a chevron or accordion children.
3. `Organisation` must remain an accordion parent row with nested `Tenant`, `Industry`, and `Company` links.
4. The `Organisation` parent row must not show selected/active highlight when a child route is active.
5. Only the active nested child row may show selected highlight.
6. The accordion must keep smooth open and close motion using height/grid, opacity, transform, and chevron rotation transitions.
7. Sidebar menu item icons must remain plain icons without boxed background, border, or shadow unless a future rule explicitly replaces this locked style.
